Toggle categories on/off to isolate what you're tracing. Same IC markers as bottom_pcb_chip_map.html, over a different base: pic/bottom_pcb_backlit_chipmap_base.jpg — the backlit both-copper-layers image (magenta = solder side, green = component side) warped into this chip map's exact pixel frame, so a marker sits on the same chip while you see both trace layers under it. Registration (component photo → rig photo via SIFT, → backlit via the verified chain) carries a few px of residual — good for seeing which layer a trace runs on, not for pin-level certainty. Naming convention: <part>-<board><n>, board B = bottom/CPU board, T = top/front-panel board.
